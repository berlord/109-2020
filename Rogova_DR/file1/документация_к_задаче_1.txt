Корректный ввод: " ./a.out name " (где name - это имя директории для старта поиска); при некорректном вводе будет выведено сообщение с просьюой повторить ввод.

Вывод: имена файлов, в которых найдется заданная строка (по одному имени в строке); строка, кторая должна содержаться в файле, задается внутри фунции main 
(присваивается переменной sym, по умолчанию sym = " ")

Краткое описание дейсвия:
Проверяет, что ftw подала именно файл, открывает файл, считывает в цикле строки из файла и в каждой считанной строке проверяет наличие "sym", если он найден, то переменная "ind" меняет значение с 0 на 1 и осуществляется выход из цикла. После чего, печатается имя файла, если проверка прошла успешно.
